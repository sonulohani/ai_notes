    $w_0$ + $w^T$$x$ = 0 $\Rightarrow$ $\Pi$

    If passing though origin then $w_0$ will become zero.

    If $\Pi_n$: $w^T$$x$ = 0 then $||$w$||$.$||$x$||$$cos \theta_w,_x$ = 0
    
    $w$ $\perp$$x$ $\Rightarrow$ $cos \theta_w,_x$ = $90 \degree$
